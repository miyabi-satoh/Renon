■はじめに■

Recollection of "Kanon" (以下、"Renon") は Key より発売の Kanon のゲームデータを利用したアプリケーションです。
コンセプトは回想です。
"あのシーンをもう一度見たい"― Renon は、そんなアナタにぴったりのソフトです。

主な特徴は、

  • Susie Plug-in を利用するので、画像データをそのまま利用できる。
  • 独自のスクリプトでシナリオを記述することにより、オリジナルストーリーも可能。
  • BGM に WAV ファイルを使用することができるので、CD を入れてなくてもOK 。
  • BGMにMP3ファイル等を使用することができるので、CDを入れてなくてもOK。
  • 自動文章送りを実現！　まるで映画を見ているような気分になれます。
と言ったところです。

■インストール■

必要とする環境

＊注：基本的に作者の開発環境以外での動作確認はしていません。

  • 日本語版Windows98
  • Windows Vista Home Premium SP2で開発・動作確認
  • Kanon 製品版がインストールされている （または、必要なデータがハードディスク上にある）。
  • Kanon PDT ファイルを展開できるSusie Plug-in （IFPDT10.SPIで動作確認）。

インストール方法

  1. ダウンロードしたzipファイル(RenonXXX.zip)を、適当なフォルダに展開します。
  2. Renon.exeと同じフォルダに Susie Plug-in をコピーします。
  3. BGMにMP3ファイル等を使用する場合は、BGMフォルダにコピーします。
    その際、ファイル名にはトラック番号を含めるようにしてください。 （02.mp3, Track02.mp3など）

■アンインストール■

展開したフォルダごと削除してください。
（レジストリは使用していません）

■使い方■

  1. Renon.exe を起動します。
  2. 初回起動時は「初期設定を行ってください。」のメッセージが表示されます。
    OKボタンを押すと環境設定ダイアログが表示されるので、 設定の項目を参考にして設定を行ってください。
  3. "Recollection of ..." のロゴが表示されたあと、タイトル画面が表示されます。
    • 最初から
      ファイル選択ダイアログが表示されるので、読み込むシナリオを選択します。
    • 続きから
      前回読み込んだシナリオがあれば、それを読みます。
    • 環境設定
      環境設定ダイアログを表示します。 詳しくは設定の項目を参照して下さい。
    • おまけ
      バージョン情報ダイアログを表示します。
    • 終了する
      アプリケーションを終了します。

ウィンドウメニューは直感的に分かると思うので、説明は省略します。

■設定■

  • "Kanon"のデータフォルダ
    Kanon がインストールされているフォルダを指定します。
  • フォント
    文章表示に使用するフォントを指定します。
  • プレイヤーの名前
    あなたの本名を入力します（任意）。

例

C:\Key\Kanon に Kanon がインストールされている場合、以下の様なフォルダ構成になっています。 

C:\
 + Key\
   + Kanon\
       + DAT\
       + PDT\
       + AVG3216M.exe
       + …

この場合、「"Kanon"のデータフォルダ」にはC:\Key\Kanon を選択します。

名前の秘密。
実は、 "Recollection of Kanon" は後付けで、先に Renon という名前が決まっていたんですね。
Kanon と Renon。
スペルの見た目が似ているのと、"Re" という接頭語が"再び"とか"もう一度"を意味するってあたりから命名。
んで、英和辞書で Re- を探したら、"Recollection" って単語があったんで、"Recollection of Kanon" になったんです。
REcollectioN Of kanoN？
RecollEction of kaNON？
真相は闇の中です、作者にも分かりません（笑）。
